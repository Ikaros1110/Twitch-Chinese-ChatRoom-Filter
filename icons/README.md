# Icons Directory

This directory contains the extension icons.

## Icon Files
- icon16.png - 16x16 icon for extension toolbar
- icon32.png - 32x32 icon 
- icon48.png - 48x48 icon for extension management
- icon128.png - 128x128 icon for Chrome Web Store

## Note
Currently using placeholder icon.svg. To create actual PNG icons, you can:
1. Use an online SVG to PNG converter to convert icon.svg to different sizes
2. Use design software like GIMP, Photoshop, or Figma
3. Use command-line tools like ImageMagick: `convert icon.svg -resize 16x16 icon16.png`

The icon features the Chinese character "中" (middle/Chinese) on a purple background matching Twitch's brand color (#9147ff).
